  • SELECT * FROM Customers;

    • Finds all customers, since no "FROM" is specified. In general it simply selects data from the database and shows it.

  • SELECT * FROM Customers WHERE Country='USA';

    • Finds all Customers from the USA, acts like a filter, showing all data within the specified criteria.

  • SELECT COUNT(*) FROM Products;

    • The count shows in numbers how many rows have what has been specified, in this case the Products rows.

  • INSERT INTO Customers (CustomerName, ContactName, Address, City, PostalCode, Country) VALUES ('Cardinal','Tom B. Erichsen','Skagen 21','Stavanger','4006','Norway');

    • The insert into command creates new records, in this case customer info.

  • UPDATE Customers SET ContactName='Alfred Schmidt', City='Frankfurt' WHERE CustomerID=1;

    • Like the command name implies, it updates already existing records, so in this case it updates the first customer's name and city, as the ID is 1.
